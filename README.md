# PIT WALL

Seguimiento de la Fórmula 1 desde el año 2000 hasta la actualidad. Standings de pilotos y constructores, calendario de carreras, comparador y estadísticas por temporada.

## Stack

- **Tailwind CSS** vía CDN
- **Chart.js 4.4.1** para gráficos de progresión de puntos
- **Google Fonts**: Space Grotesk + Inter + Material Symbols
- Sin build step, sin backend — hosting estático vía GitHub Pages
